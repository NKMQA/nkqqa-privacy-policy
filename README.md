# NKMQA Privacy Policy

This repository hosts the **Privacy Policy** for the **NKMQA Bug Capture Chrome Extension**.

## About the Extension

NKMQA Bug Capture is a Chrome extension designed for QA testers to capture bug notes and screenshots.

- **Data Storage:** All bug reports (notes, page URLs, and screenshots) are stored **locally** on the user’s device.  
- **No Data Sent:** The extension does **not** send any data to servers or third-party services.  
- **User Control:** Users can delete individual bug reports or clear all data at any time.

## Purpose of this Repository

This repository provides a **publicly accessible Privacy Policy URL** to meet Chrome Web Store requirements.
